# -MATLAB-
MATLAB的多目标物品自动计数与位置标注系统
实现零件、硬币、零散物料自动识别、数量统计、中心坐标提取、目标框选标注
 
一、项目介绍
在工业流水线物料清点、五金零件分拣、小件统计场景中，人工清点效率低、出错率高。本项目不使用深度学习，依靠传统数字图像处理算法，基于MATLAB搭建一套轻量化目标检测计数程序。
整体流程：图像灰度化→高斯滤波降噪→自适应阈值二值分割→形态学开闭运算净化图像→连通域特征提取→面积阈值滤除噪声→目标框选与坐标输出。
本项目全程使用课堂所学基础算法，稳定性高、运行速度快，可以直接移植到嵌入式视觉设备，适用于平面俯视下无严重粘连的小件物料检测。
 
二、开发环境
软件：MATLAB R2022b及以上，Image Processing Toolbox图像处理工具箱
系统：Windows10/Windows11
适用场景：硬币计数、机械零件清点、颗粒物料统计
 
三、算法整体流程
1.图像预处理：彩色图像灰度转换，高斯滤波消除拍摄高斯噪声，消除画面颗粒感
2.图像分割：采用自适应阈值二值化，解决光照不均、局部阴影造成的分割失效问题，将物体转为白色前景，背景转为黑色
3.形态学处理：开运算去除粉尘、光斑、细小噪点；闭运算填充物体内部孔洞，修复断裂轮廓，保证每个零件为完整连通域
4.连通域分析：bwlabel标记所有连通区域，regionprops提取每个区域的面积、中心坐标、外接矩形参数
5.噪声筛选：设置像素面积阈值，过滤灰尘、杂点产生的无效连通域
6.可视化输出：在原图绘制彩色包围框、中心点、目标编号，控制台输出物体总数与坐标信息

五、运行方法
1.将main.m与测试图片test_parts.jpg放在同一文件夹
2.使用MATLAB打开脚本，直接点击运行
3.程序会自动弹出六张子图，依次展示原图、灰度图、滤波图、二值图、形态学处理图、最终标注结果
4.命令行窗口自动输出目标总数量、每个物体的中心坐标、外接矩形参数
 
六、核心可调参数
imgaussfilt(gray,1.2)：高斯滤波强度，画面噪点多时增大数值
strel('disk',2)：开运算结构元，噪点过多可以增大半径
strel('disk',3)：闭运算结构元，物体出现断裂、孔洞可增大半径
area_thresh = 150：面积阈值，用来过滤微小噪声，零件尺寸越小，阈值设置越低
 
七、算法优缺点
优点：无需深度学习、运算速度快、代码模块化、可二次开发、硬件要求极低、能够输出坐标数据
缺点：物体粘连时会合并为单个连通域，出现漏检；强光过曝环境识别精度下降；仅支持俯视平面拍摄
 
八、优化拓展方向
1.引入分水岭算法，分割相互粘连的零件，提升粘连物体识别准确率
2.增加直方图均衡化，解决强光、阴影干扰问题
3.添加圆形度筛选，可以区分圆形硬币与不规则杂质
4.封装为GUI界面，做成可视化软件
